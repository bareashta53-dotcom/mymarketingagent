# Facebook Ads Bot Setup Guide

כדי שהבוט שלנו יוכל לתקשר עם מנהל המודעות שלך (Meta Ads Manager), אנחנו צריכים להגדיר לו גישה ולהתקין את סביבת העבודה. 
עקוב אחר השלבים הבאים:

## שלב 1: התקנת פייתון (Python)
המחשב שלך כרגע ללא פייתון מותקן, ולכן הבוט לא יכול לרוץ.
1. היכנס לחנות האפליקציות של ווינדוס (Microsoft Store).
2. חפש "Python 3.12" (או הגרסה הכי חדשה שמופיעה).
3. לחץ על "התקן" (Install).
4. לחלופין, אפשר להוריד מהאתר הרשמי: https://www.python.org/downloads/

## שלב 2: יצירת גישה ב-Meta (פייסבוק)
כדי שהקוד שלנו ידע לגשת לחשבון שלך, נדרש מפתח הצפנה.
1. היכנס לאתר המפתחים של מטא: [https://developers.facebook.com](https://developers.facebook.com)
2. התחבר עם המשתמש פייסבוק שלך שיש לו הרשאות למנהל המודעות.
3. לחץ על הלחצן **"My Apps"** (בצד ימין למעלה) ואז על **"Create App"**.
4. בחר באפשרות **"Other"** ואז **"Business"**. תן שם לאפליקציה (למשל Marketing Bot).
5. בתוך האפליקציה, חפש את המוצר **"Marketing API"** ולחץ על **"Set Up"**.
6. בתפריט הצדדי, תחת Marketing API, לחץ על **"Tools"** (כלים).
7. סמן את ההרשאות: `ads_management` ו-`ads_read`.
8. לחץ על הכפתור **"Get Token"** (קבל מפתח).
9. **העתק את רצף האותיות והמספרים הארוך שקיבלת.**

## שלב 3: הגדרת מזהה החשבון (Ad Account ID)
1. היכנס למנהל המודעות שלך בפייסבוק (Ads Manager).
2. בשורת הכתובת (URL) או בפינה השמאלית העליונה, תראה מספר חשבון (למשל: 1234567890).

## שלב 4: חיבור הנתונים לבוט
1. פתח את הקובץ `.env` שנמצא בתיקייה זו.
2. בשורה הראשונה, הדבק את המפתח הארוך במקום `your_access_token_here`.
3. בשורה השנייה, הדבק את מספר החשבון שלך *אחרי* הקידומת `act_` (למשל: `act_1234567890`).

ברגע שתסיים את השלבים האלו – עדכן אותי פה בצ'אט ונוכל להריץ את הבוט ולבדוק שהחיבור עובד!
